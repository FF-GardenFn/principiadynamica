# Visualization API Reference

The `northstar.vis` module provides visualization tools for alignment trajectories, metrics, and analysis results.

## TrajectoryVisualizer

```python
class TrajectoryVisualizer(use_rich: bool = None, width: int = 60)
```

Visualizes alignment trajectories and metrics in the console.

This class provides methods for creating sparklines, tables, and other visualizations of alignment data.

**Parameters**:
- **use_rich** (`bool`, optional): Whether to use rich formatting (None = auto-detect)
- **width** (`int`, default=60): Default width for visualizations

### Methods

#### `visualize_alignment_history`

```python
visualize_alignment_history(alignment_scores: List[float], width: Optional[int] = None, threshold: float = 0.7) -> None
```

Visualize alignment history as a sparkline.

**Parameters**:
- **alignment_scores** (`List[float]`): List of alignment scores
- **width** (`int`, optional): Width of the visualization (defaults to self.width)
- **threshold** (`float`, default=0.7): Alignment threshold to highlight

**Example Output**:
```
Alignment Trajectory:
█████▆▅▅▄▃▂▂▁▁
Start: 0.85 → End: 0.45
```

#### `print_trajectory_analysis`

```python
print_trajectory_analysis(analysis: Dict[str, Any], predictions: Optional[List[Dict[str, Any]]] = None) -> None
```

Print trajectory analysis results.

**Parameters**:
- **analysis** (`Dict[str, Any]`): Stability metrics dictionary
- **predictions** (`List[Dict[str, Any]]`, optional): Optional trajectory predictions

**Example Output**:
```
Trajectory Analysis
------------------
Average Alignment: 0.723
Alignment Trend: -0.150
Alignment Volatility: 0.082
Stability Score: 0.918

✓ The system exhibits strong alignment stability

Trajectory Predictions
---------------------
Step 1: Alignment = 0.720 (→ 0.000)
Step 2: Alignment = 0.715 (↓ 0.005)
Step 3: Alignment = 0.708 (↓ 0.007)
Step 4: Alignment = 0.700 (↓ 0.008)
Step 5: Alignment = 0.690 (↓ 0.010)

✓ The system is predicted to remain aligned
```

#### `print_robustness_analysis`

```python
print_robustness_analysis(robustness: Dict[str, Any]) -> None
```

Print robustness analysis results.

**Parameters**:
- **robustness** (`Dict[str, Any]`): Robustness metrics dictionary

**Example Output**:
```
Robustness Analysis
------------------
Base Alignment: 0.750
Average Change: -0.025
Max Negative Change: -0.120
Max Positive Change: 0.050
Robustness Score: 0.850
PSD Distance: 0.120

✓ The system exhibits strong alignment robustness
```

#### `print_optimization_results`

```python
print_optimization_results(results: Dict[str, Any]) -> None
```

Print optimization results.

**Parameters**:
- **results** (`Dict[str, Any]`): Optimization results dictionary

**Example Output**:
```
Optimization Results
-------------------
Energy: 0.1234
Solver: dwave

Optimized Path:
1. state_1 (alignment_score: 0.8)
2. state_3 (alignment_score: 0.75)
3. state_7 (alignment_score: 0.82)
```

## Visualization Helpers

### `create_visualizer`

```python
create_visualizer(use_rich: bool = None, width: int = 60) -> TrajectoryVisualizer
```

Create a new TrajectoryVisualizer instance.

**Parameters**:
- **use_rich** (`bool`, optional): Whether to use rich formatting (None = auto-detect)
- **width** (`int`, default=60): Default width for visualizations

**Returns**:
- `TrajectoryVisualizer`: TrajectoryVisualizer instance

**Example**:
```python
visualizer = create_visualizer(use_rich=True, width=80)
visualizer.visualize_alignment_history(alignment_scores)
```

## Rich vs. Plain Text Output

The visualization module supports two output modes:

1. **Rich Mode**: Uses the [Rich](https://github.com/Textualize/rich) library to create colorful, formatted console output with tables, progress bars, and colored sparklines.

2. **Plain Text Mode**: Falls back to plain text output when Rich is not available or when explicitly disabled.
